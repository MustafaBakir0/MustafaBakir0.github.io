Library Catalog Management System (LCMS)

C++ Development / Data Structures

Type

University Coursework

Year

2025

Role

Design, Development

Technologies

C++, Data Structures, OOP

The Library Catalog Management System (LCMS) is a C++ application designed to efficiently organize and manage books, categories, and borrowers in a library setting. Built with object-oriented principles, this system allows books to be stored in a hierarchical category structure (e.g., Fiction/Sci-Fi or Non-Fiction/History), enabling intuitive navigation and management. The program supports comprehensive book operations, including adding, editing, removing, and searching for books by title, as well as importing and exporting catalog data via CSV files.

At its core, the system leverages a custom tree data structure to maintain the category hierarchy, with each node tracking its book count and subcategories. For dynamic storage, I implemented MyVector, a custom vector class with iterator support, dynamic resizing, and efficient element management. The application also handles borrower management, tracking current and past book loans while providing functionalities for borrowing, returning, and listing borrowed books per user.

Key Features

  • Hierarchical Category Structure: Organizes books in an intuitive tree-based system
  • Custom Vector Implementation: Dynamic storage with iterator support
  • Borrower Management: Tracks loans and returns with user-specific history
  • CSV Import/Export: Facilitates data migration and backup

Development Process

This project highlights my proficiency in C++ memory management, custom data structure design, and algorithmic problem-solving. The use of destructors for resource cleanup, operator overloading for book comparisons, and recursive tree traversal demonstrates strong software engineering practices. The command-line interface ensures ease of use, making it a practical solution for library catalog management.

Results & Impact

Designed to showcase my ability to develop structured, scalable systems, this project is an excellent addition to my portfolio, emphasizing C++ expertise, efficient data organization, and real-world application development. The system demonstrates how proper data structure selection and implementation can create an efficient, user-friendly library management solution that handles complex hierarchical relationships and data operations.
